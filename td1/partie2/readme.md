## Commandes pour lancer le serveur et le client

### Lancer le serveur

flask run

### Lancer le client

mettre le chemin du fichier quiz.html dans le navigateur

## Commandes curl pour interagir avec le serveur

http://localhost:5000/quiz/api/v1.0/quiz
